# Details
* Font Used : brass mono
* GTK Theme : Adapta Blue
* Icons     : Custom Version of papirus
* OS        : Alter Linux
* Terminal  : Kitty
* Shell     : zsh with [Starship Prompt](https://starship.rs/) 
* Bar       : Polybar
